# Arielo MiniPC OS - 10AJ TactileBrowser GUI Graphical Selectors

Pruebas recomendadas:

```text
desktop
WEB
```

Dentro del navegador:

```text
FILE  -> selector grafico de archivos (/root, /sdcard, /usb)
FAVS  -> selector grafico de favoritos
SAVE  -> guardar pagina actual en favoritos
LINK  -> overlay/lista de enlaces de la pagina actual
BACK  -> atras
HOME  -> about:home
HELP  -> ayuda
EXIT  -> salir
```

Tambien desde consola:

```text
tbbgui files
tbbgui bookmarks
tbbgui about:files:/sdcard
```
