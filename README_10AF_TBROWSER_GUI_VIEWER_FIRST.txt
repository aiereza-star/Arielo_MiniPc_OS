Arielo MiniPC OS - 10AF_TBROWSER_GUI_VIEWER_FIRST

Base: 10AE_FIX1_BOOKMARKS_SAFE_COPY validada.

Objetivo 10AF:
- Primer visor grafico prudente para TactileBrowser.
- Lexbor integrado en firmware, PSRAM-first.
- Reutiliza motor HTML/local/enlaces/favoritos de 10AE.
- Nuevo comando: tbbgui / tbbrowsergui.
- tbbrowser queda como fallback texto 10AE validado.

Controles GUI:
- n / ENTER / espacio: pagina abajo
- p: pagina arriba
- j/k: linea abajo/arriba
- g/G: inicio/final
- 1-9: abrir enlace visible por numero
- b: atras
- r: recargar
- l: overlay de enlaces
- h / ?: overlay de ayuda
- m: guardar favorito actual
- q: salir

Nota:
Esta es la primera carroceria grafica. Sin CSS, imagenes, red ni formularios.
