# Arielo Breakout 10B

Fix contra parpadeo: usa SM_400X240 y dibujo delta, sin limpiar toda la pantalla cada frame.

## Compilar
```bat
buildelf.bat
```

## Copiar
`breakout.xtensa.elf` a `/root/bin`.

## Controles
- Izquierda/derecha o A/D: mover pala
- P/Espacio: pausa
- R: reiniciar
- Q/ESC: salir
