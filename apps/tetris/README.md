# Arielo Tetris 10B

Fix contra reinicio: version rehecha en SM_400X240, siguiendo el patron estable de Snake/Pong.

## Compilar
```bat
buildelf.bat
```

## Copiar
`tetris.xtensa.elf` a `/root/bin`.

## Controles
- Izquierda/derecha o A/D: mover
- Arriba/W/Z/C: rotar
- Abajo/S: caida rapida
- X/V/Espacio: caida instantanea
- P: pausa
- R: reiniciar
- Q/ESC: salir
