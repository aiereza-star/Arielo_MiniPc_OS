Arielo MiniPC OS - 10AG HOME WEB BUTTON LEXBOR GUI

Objetivo:
Reemplazar el boton WEB del escritorio por el nuevo TactileBrowser grafico con Lexbor integrado.

Comportamiento:
- En escritorio, tarjeta WEB / Lexbor GUI.
- Al pulsarla, se ejecuta: tbbgui file /root/tbbrowser_home.html
- Si no existe /root/tbbrowser_home.html, se crea una pagina inicial local.
- Al salir de tbbgui con q, vuelve al escritorio.

No se elimina fisicamente el navegador antiguo; queda como respaldo dentro del codigo.
