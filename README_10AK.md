# WAVESHARE_7_MINIPC_BREEZYBOX_LAB_10AK_TBROWSER_GUI_VISUAL_RENDER_POLISH

Escalon 10AK del Arielo MiniPC OS.

La 10AK no agrega motor nuevo: pule la vista grafica del TactileBrowser ya validado.

Comandos:

```text
tbbgui home
tbbgui files
tbbgui bookmarks
tbbgui file /sdcard/test_10ak_visual.html
```

Desde escritorio:

```text
desktop -> WEB
```

Controles principales:

```text
BACK HOME FILE FAVS SAVE LINK HELP EXIT
```

Criterio de validacion:
- Que abra la pagina HOME interna.
- Que abra archivos desde FILE.
- Que favoritos FAVS sigan funcionando.
- Que los enlaces sigan abriendo.
- Que visualmente se distingan H1/H2/listas/enlaces.
- Que no haya reinicios.
