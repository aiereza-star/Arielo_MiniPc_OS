# 10AL_FIX2_HEADER_NAVBAR_INLINE_URL

Base: 10AL_FIX1_HTTP_BUILD_AND_LOAD.

Cambios:
- La barra URL deja de abrirse como ventana modal central.
- Nueva barra de navegación integrada en la cabecera del navegador gráfico.
- La cabecera muestra título + campo URL/ruta siempre visible.
- Al pulsar URL o tecla `u`, la edición se hace sobre esa misma cabecera.
- Se conserva HTTP básico, FILE/FAVS/SAVE/LINK/BACK/HOME/EXIT.

Objetivo: que el navegador tenga aspecto más real y menos de diálogo flotante.
