# Arielo MiniPC OS - TactileBrowser 10AQ

Versión de pulido sobre `10AP_OK_TBROWSER_DDG_HTML_SEARCH`.

Objetivo: hacer más práctico el buscador DuckDuckGo HTML sin meter JavaScript, CSS ni imágenes.

## Uso

Desde el escritorio:

```text
WEB
```

En la barra URL se puede escribir:

```text
ddg: esp32 s3 waveshare
buscar arduino psram
?orange pi zero 2w tdt
https://google.com/
http://neverssl.com/
```

## Novedad 10AQ

- Resultados DDG HTML más limpios.
- Overlay `LINK` más útil.
- Decodificación de enlaces DuckDuckGo `/l/?uddg=...` para abrir directamente el destino real.

## Salva recomendada si valida

```text
10AQ_OK_TBROWSER_DDG_RESULTS_POLISH
```
